What is this?

Keyless is a static web app that talks to public, no-auth LLM endpoints. There is no backend. There are no API keys. There is no tracking. The whole thing is a single HTML page served from GitHub Pages.

How is it free?

  • Pollinations.ai — invites public use of their inference API. Bandwidth paid by their sponsors.
  • DuckDuckGo AI Chat — keyless after a two-step VQD handshake, routed through a CORS relay.
  • WebLLM — runs open models locally in your browser via WebGPU. Zero network cost, zero privacy exposure.

If one provider fails?

Set provider to auto and the router retries the next one until something answers.

Will this always work?

Public endpoints change. If everything upstream breaks, switch to webllm and the app still runs — the model downloads once, then inference is fully local.
