What is this?

KeylessAI is a static web app that talks to public, no-auth LLM endpoints. There is no backend. There are no API keys. There is no tracking. The whole thing is a single HTML page served from GitHub Pages.

How is it free?

  • Pollinations.ai invites public use of their inference API. Bandwidth paid by their sponsors.
  • WebLLM runs open models locally in your browser via WebGPU. Zero network cost, full privacy.
